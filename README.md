# TODO LIST
Application developed with React to practice useState, arrays, saving data in the browser cache and updating the DOM

## Characteristics:
1.- Updated date and month
2.- Responsive container for stored tasks
3.- Task Creation
4.- Delete Tasks
5.- Complete tasks
6.- Responsive Design
7.- Motion Effects
